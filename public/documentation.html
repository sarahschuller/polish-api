Nailpolish API

A web application that allows users to create an account and save Nailpolishes to their library list as well as a favorites list and wishlist. Users can also update their profile and remove polishes from their given lists.

Features

Request Type URL Returns Example Message
Returns a list of all nailpolishes to the users GET /polishes/:name An Array of polishes within the database as a JSON object N/A N/A
Return data about polishes by color family GET /colorfamlies An array of polishes by color family as a JSON object N/A N/A
Return data about polishes by finish GET /finishes An array of polishes by finish as a JSON object N/A N/A
Register new users POST /users Create a JSON object with data about a newly registered user { name: "John Doe" username: "johndoe123" email: "johndoe@example.com" password: "abcd123" birthday: "01/01/01" } New User has been registered
Users are able to update their information PUT /users/:username N/A { name: "John Doe" username: "johndoe123" email: "johndoe@example.com" password: "efgh456" birthday: "01/01/01" } User information has been updated
Allow users to add a polish to their favorites list POST /users/:username/polishes/:polishId Create a JSON object with data about a user's favorite polishes N/A polish has been added to user favorites
Allow users to remove polishes from their favorites list DELETE /users/:username/polishes/:polishId Removes an item from the JSON object with the user's list of favorite polishes N/A polish has been removed from favorites

Allow users to remove polishes from their wishlist DELETE /users/:username/polishes/:polishId Removes an item from the JSON object with the user's list of wishlist polishes N/A polish has been removed from wishlist
Allow current users to delete their registration DELETE /users Removes a user account from the database N/A Your account was successfully deleted
